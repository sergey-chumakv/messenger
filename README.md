Проект messenger:  
Для обмена сообщениями между пользователями.

Запуск проекта: npm run start  
Сборка проекта: npm run build  
Запуск тестов: npm run test  
Проверка типов: npm run check-types  

Стек:  
TypeScript,  
SCSS,  
Handlebars,  
Webpack,  
Mocha
